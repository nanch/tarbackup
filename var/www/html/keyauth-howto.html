key-based authentication with tarbackup

Create your public/private key

# cd ~
# mkdir .ssh
# cd .ssh
# ssh-keygen -f tarbackup_rsa -C 'account@tarbackup.com' -N '' -t rsa -q

Upload your public key to tarbackup

Go to https://tarbackup.com/addkey.html to upload your public key (tarbackup_rsa.pub).

Setup your local config

Paste this code snippet in your shell, and replace USERNAME with your tarbackup username.
# cat >> ~/.ssh/config << CONFIG_SNIPPET
Host tarbackup.com
    User USERNAME
    IdentityFile ~/.ssh/tarbackup_rsa
CONFIG_SNIPPET

Use sftp normally

Now you need to supply neither your password nor your username, so you can just supply this command:
# sftp tarbackup.com
